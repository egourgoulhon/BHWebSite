If you wish to download the sources and build the pdf by yourself, run the following commands in a terminal: 

        git clone https://github.com/egourgoulhon/BHLectures.git
        cd BHLectures/Notes
        pdflatex bholes
        pdflatex bholes
        pdflatex bholes
        makeindex bholes
        pdflatex bholes
